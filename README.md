# Minecraft-Bedrock-Launcher-And-Client

A curated list of 3rd party launchers, clients, and tools for Minecraft Bedrock Edition (Windows & Android).

## Windows Launchers

| Name | Description | Features | Source |
| :--- | :--- | :--- | :--- |
| **Amethyst Launcher** | A modding-focused launcher for MCBE. | Native C++ mod support, version management, and profile isolation. | [GitHub](https://github.com/FrederoxDev/Amethyst-Launcher) / [Discord](https://discord.com/invite/q8k3CmqP46) |
| **Bedrock Launcher** | An unofficial, Java-like launcher for Windows. | Profile management, version switching, and separate data folders. | [GitHub](https://github.com/BedrockLauncher/BedrockLauncher) / [Website](https://bedrocklauncher.github.io/) |
| **MCLauncher** | A multi-version launcher for Windows 10/11. | Simple side-by-side version installation and UWP/GDK support. | [GitHub](https://github.com/MCMrARM/mc-w10-version-launcher) |
| **Levi Launcher (Windows)** | A GDK-based launcher for Windows. | Multi-version management, version isolation, mods etc. | [GitHub](https://github.com/LiteLDev/LeviLauncher) / [Discord](https://discord.gg/fkgsQZeRQV) |
| **Jiayi Launcher** | A legacy mod manager and version switcher for Windows. | Renderdragon shader support, UI customizability, and version switching. | [GitHub](https://github.com/JiayiSoftware/JiayiLauncher) / [Discord](https://discord.gg/vMYpA6UnKz) |
| **Project Star** | A C#-based recreation of the Astral Launcher aimed at stability. | Clean UI, Version manager, and geometry skin pack integration. | [Discord](https://discord.gg/nBMvmDdjsg) |
| **Vanilla RTX App** | A specialized tool for Minecraft Ray Tracing. | Automatic resource updates, one-click RTX launcher, DLSS swapper, and BetterRTX manager. | [MS Store](https://apps.microsoft.com/detail/9n6pcrz5v9dj) / [GitHub](https://github.com/Cubeir/Vanilla-RTX-App) |

## Android Launchers

| Name | Description | Features | Source |
| :--- | :--- | :--- | :--- |
| **Atlas Client** | The best client for MCPE. | FPS Boost, custom UI, and various utility mods for Android. | [Website](https://atlasclient.net/) / [Discord](https://discord.gg/atlas-client-1177368741705695292) |
| **Levi Launcher (Android)** | A dedicated Bedrock launcher for Android. | Multi-version management, version isolation, mods, multiple xbox account etc. | [GitHub](https://github.com/LiteLDev/LeviLaunchroid) / [Discord](https://discord.gg/fkgsQZeRQV) |
| **MB Loader** | A lightweight utility for running Minecraft on Android. | Support shader, backup & restore data, Q button etc. | [Google Play](https://play.google.com/store/apps/details?id=io.bambosan.mbloader) / [Discord](https://discord.gg/FbSYdPtBYm) |
| **BlockLauncher** | A launcher that patches Minecraft for Android. | Custom textures, skins, and ModPE scripts. (**Discontinued**) | [GitHub](https://github.com/zhuowei/MCPELauncher) |

## Clients

| Name | Description | Features | Source |
| :--- | :--- | :--- | :--- |
| **Latite** | A popular DLL-based modification for Windows. | Clean UI, 30+ customizable mods, and a JavaScript/TypeScript plugin system. | [GitHub](https://github.com/marioCST/Latite) / [Discord](https://discord.gg/RspeR2MHqQ) / [Website](https://latite.net/) |
| **OderSo** | A legitimate PvP-focused client for Bedrock. | High performance, customizable HUD, and regular version support. | [Discord](https://discord.gg/kVPFvgPmTC) / [Website](https://cdn.odersoclient.com/)|
| **Flarial** | A feature-rich client inspired by Java's Lunar Client. | Custom UI, FPS optimization, Motion Blur, and Keystrokes. | [Website](https://flarial.xyz/) / [GitHub](https://github.com/flarialmc) / [Discord](https://discord.gg/R82VYmtCt9) |
| **Onix Client** | One of the most long-standing Bedrock clients. | Item physics, scripting support (paid), and extensive module list. | [Website](https://onixclient.com/) / [Discord](https://discord.gg/onixclient) |
| **VoidLine** | A utility-heavy client for Windows 10/11. | Innovative "Auto Map Vote" for Hive, FPS presets, and modular UI. | [Discord](https://discord.gg/8KZ3qrw5FR) |
